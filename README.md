## 项目简介

量子计算实验对环境条件极为敏感，微小的温度波动或振动干扰都可能导致实验失败。q-lab-monitor 旨在为量子实验室提供一套轻量级、实时、可视化的环境监控解决方案，支持：

- 多传感器数据采集与聚合
- 实时数据展示与历史趋势分析
- 异常阈值告警（邮件/短信/Webhook）
- 数据持久化存储与导出

## 快速开始

### 环境要求

- JDK 17+
- Maven 3.9+
- MySQL 8.0+
- Redis 7+
- MQTT Broker (可选)

### 安装步骤

1. 克隆项目

   ```bash
   git clone https://github.com/LongGuann/q-lab-monitor.git
   cd q-lab-monitor
   ```

2. 创建数据库

   ```bash
   mysql -u root -p < sql/init.sql
   ```

3. 修改数据库配置

   编辑 `src/main/resources/application.yml`，将数据库密码改为实际密码。

4. 启动后端

   ```bash
   mvn spring-boot:run
   ```

5. 启动前端

   ```bash
   cd frontend
   npm install
   npm run dev
   ```

6. 访问系统

   浏览器打开 `http://localhost:8080`
